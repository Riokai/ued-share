# 商派UED分享主题-Grunt

## 准备工作

通过NPM全局安装grunt-cli插件
```shell
npm install -g grunt-cli
```

切换到ued-share目录，安装项目依赖插件
```shell
npm install
```

## 演示内容

### demo01
watch和connect插件实现实时刷新页面

```shell
grunt demo01
```

### demo02
watch、sass、autoprefixer打造CSS开发环境

```shell
grunt demo02
```

### demo03
uglify插件实现JavaScript代码压缩，合并

### demo04
spritesmith自动合并精灵图
